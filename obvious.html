Obvious New Website

Redesigning the digital presence with modern aesthetics and improved user experience.

Project Overview

Obvious is a creative agency that needed a complete digital transformation. This project involved redesigning their entire web presence to reflect their innovative approach and showcase their portfolio in a more engaging way.

Key Features

  • Modern, minimalist design aesthetic
  • Improved navigation and user flow
  • Portfolio showcase with filtering
  • Responsive design for all devices
  • Enhanced content management system

Design Process

The redesign focused on creating a clean, modern interface that would better represent the agency's creative capabilities while improving the overall user experience for potential clients and visitors.

Technologies Used

  • Figma for design and prototyping
  • React for frontend development
  • Contentful for content management
  • Netlify for hosting and deployment

Results

The new website resulted in a 60% increase in user engagement and a 40% improvement in lead generation. The modern design better represents the agency's creative capabilities and provides a superior user experience.
