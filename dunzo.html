Dunzo

Revolutionizing hyperlocal delivery with seamless user experience

Client Dunzo
Year 2023
Role UX Designer

Overview

Dunzo is India's leading hyperlocal delivery platform, connecting users with local stores and delivery partners. The project focused on redesigning the core ordering experience to make it faster, more intuitive, and more reliable.

Our goal was to reduce order completion time while improving user satisfaction and increasing order frequency.

The Challenge

Dunzo needed to streamline their ordering process while maintaining the personal touch that users love. The main challenges were:

  • Reduce order completion time from 3 minutes to under 90 seconds
  • Improve order accuracy and reduce customer support tickets
  • Create a more engaging tracking experience
  • Optimize for both new and power users

Process

Discovery

Analyzed user behavior data, conducted interviews with frequent users, and mapped the current ordering journey to identify pain points.

Design

Created wireframes and prototypes focusing on speed and clarity, with special attention to the checkout flow and tracking experience.

Testing

Conducted A/B testing with real users to validate design decisions and measure impact on order completion time.

Launch

Rolled out the new design gradually, monitoring key metrics and gathering user feedback for continuous improvement.
